2. Description of Service

IntroHater is a metadata service that provides timing information (skip segments) for video content. The Service:

  • Generates HLS manifests with byte-range instructions

  • Offers a Stremio addon for integrating skip functionality

IntroHater does not host, store, stream, download, upload, or distribute any video content. The Service only provides metadata that instructs video players on which portions of a stream to skip.

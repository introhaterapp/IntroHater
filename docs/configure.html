What is AIOstreams? (read this if you're stuck)

AIOstreams is a separate Stremio addon that finds torrent streams through your debrid service. IntroHater does not search for torrents itself — it reuses the streams AIOstreams already finds, then adds intro/credits skip on top.

What to do: Open AIOstreams, connect your debrid account, pick a scraper (Comet recommended), install it to Stremio, then copy the manifest URL it gives you and paste it below.

Step 1: Configure AIOstreams with your debrid + scraper (Comet recommended).
Step 2: Copy the AIOstreams manifest URL and paste it above.
Multi-Provider: If AIOstreams uses multiple debrid services, add each provider above. IntroHater picks the right key per stream.

📖 Setup Guide
1 Copy your AIOstreams manifest URL

Configure AIOstreams with your debrid service + Scraper (Comet currently recommended), then copy the generated URL.

2 Paste URL & API Key into IntroHater

Enter your debrid API key and paste the AIOstreams manifest URL in the form above.

3 Install IntroHater to Stremio

Click "Install on Stremio" to add the addon to your Stremio app.

4 Open a show and see IntroHater listed

Browse to any TV show or movie - you'll see IntroHater streams in the list.

5 Desktop: pick the Skip stream

On desktop, select the stream with the 🎯 icon for auto-skipped intros.

6 TV / Fire Stick: pick the right stream

Android TV / Fire Stick with MKV torrents: use the Direct stream for playback (no skip).

Desktop or MP4 sources: use the Skip stream for auto-skip.

We need your API Key to fetch streams via Torrentio.

Recommended: Use AIOstreams - configure it with your debrid service, then paste the manifest URL here.

Why AIOstreams? It runs on ElfHosted servers which aren't rate-limited (unlike Torrentio direct). Your streams will work reliably without "no results" errors.
